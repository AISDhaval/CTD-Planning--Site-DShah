Discussion

Fireworks

Pros

  • Fireworks has pages and states instead of just layers.
  • Built-in web layer.
  • Firework's ability to compress images.

Cons

  • Internal Linking
  • If you're used to traditional Adobe UIs, like Illustrator or Photoshop, you'll probably be lost, confused and continually frustrated in Fireworks as the menus are very different, keyboard shortcuts are different and things are located in completely different places in the UI.
  • Gradient banding: making gradients in Fireworks tend to not blend too well and end up looking banded. The best bet is to make a gradient in Photoshop, then bring it into Fireworks. Strange, I know, but it works.
  • It's quirky. Fireworks does random stuff all the time. It's gotten better with CS5, which fixed some 16,000 documented bugs or some insane number, but it still misbehaves from time to time. After a while, you just come to expect the little troublemaker.

Dreamweaver

Pros

  • Code Highlighting
  • Code Suggestion
  • Code and Design View
  • Tabs and Connected files

Cons

  • Bulky Dynamic Code
  • Very Expensive
  • Not in full control of Code
  • CSS is in the same sheet which may be hard to understand
